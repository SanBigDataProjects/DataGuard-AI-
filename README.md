# 🛡️ DataGuard AI
### AI-Powered Data Quality Monitor
**Built as part of the OpenAI x Handshake Codex Creator Challenge**

## 🚀 What is DataGuard AI?
DataGuard AI automatically detects data problems, calculates a health score out of 100, and uses Groq LLaMA AI to explain issues in plain English with fix recommendations.

## ✨ Features
- 🔍 Detects missing values, duplicates, outliers, and invalid data
- 📊 Calculates overall data health score out of 100
- 🤖 AI explains every issue in plain English
- 💡 Provides Python code to fix each issue
- 📈 Interactive Streamlit dashboard

## 🛠️ Tech Stack
- Python, Pandas, NumPy
- Groq LLaMA AI
- Streamlit, Plotly
- SQLite

## 🚀 How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
